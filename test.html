üîç Technical Details

The extension now uses sessionStorage to persist state across page reloads:

  • State is saved every second while timer is running
  • Timestamp is recorded to calculate elapsed time during reload
  • On page load, state is restored and timer auto-resumes if it was running
  • Explicitly closing the timer clears the session storage
